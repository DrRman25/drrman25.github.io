Setting Up a Minecraft Server

Welcome to my text-based tutorial on setting up a Minecraft server. This server is going to be completely vanilla, meaning you can't install any plugins or mods.

Things to Do Beforehand

Before you get started with this tutorial, you need to, obviously, have a desktop or laptop that meets the Minecraft: Java Edition system requirements, and you need to buy Minecraft for $29.99. This will grant you access to both Minecraft: Java Edition and Minecraft: Bedrock Edition. However, you only need Minecraft: Java Edition for this tutorial, which is available for Linux, macOS, and Windows.

You will also need to install Java 17, which is compatible with Minecraft: Java Edition 1.19.2, the latest version. If you have other version(s) of Java installed, uninstall ALL of them, and install Java 17.

The Server .jar File

The reason why you need to install Java 17 is because you need to be able to run the server .jar file, which should be named something like minecraft_server.1.19.2.jar. You can download it on the Minecraft website.

Once you have downloaded the file, create a new folder (preferably on your desktop for ease of access), and name it whatever you want. After that, put the .jar file in your folder, and double-click on it with your primary mouse button. It should create a bunch of files, and then say "Press any key to continue".
