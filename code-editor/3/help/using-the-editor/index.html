Using the Editor

This page provides a basic tutorial on using the HTML, CSS, and JavaScript editor. However, editors for different languages have similar, if not the same, features.

The Editor Interface

There are several buttons above the editor, which do different things. They will be mentioned again in a later section. For now, let's look at the editor itself.

There are two sections of the editor's interface currently. The first one is the area where you type in code you want to run live. The second one is the output area, where you actually see the output, after you click or tap the "Run" button or press "Ctrl-Enter" on your keyboard.

If you have a wide screen (a screen width greater than or equal to 1200px), both the text editor and the output area will always be present on the left and right sides of the screen, respectively. If you have a narrow screen (a screen width less than 1200px), there will be buttons at the top right to switch between the text editor and the output area. If you don't know what your screen width is, use the button below to display it in a tooltip.

The Buttons

There are normally four buttons at the top left, which you can click or tap to perform certain actions.

Run

The first button, labeled by "Run", takes the code you typed into the text editor, interprets it, and shows the output. However, there is a 500-millisecond cooldown for this button, to prevent spamming.

Share

This next button, labeled by "Share", opens a menu, where you can get a link to your program, or download the code to your computer.

Save

The third button, labeled by "Save", allows you to save your program to My Programs, a place designated for storing your valuable code.

Clear Output

The fourth button, labeled by "Clear Output", clears the output for you. This is useful in cases where you run the code, and it gets a little too hectic, or the program uses up too many of your computer's resources.
